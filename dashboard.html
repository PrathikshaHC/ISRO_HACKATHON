How to read these predictions

Probability is a combined metric derived from the model's predicted strength. Higher values indicate larger chance of impactful flares within 24 hours.

  • Model inputs: Features include position, duration, and measured activity.
  • Data source: Live flare events are pulled from NASA DONKI when available.
  • Confidence: Model-confidence reflects training-set agreement; treat it as guidance.
